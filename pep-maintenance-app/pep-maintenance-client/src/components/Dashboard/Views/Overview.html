Before you get started, please review the code base. This dashboard is based on the Vue Paper Dashboard. In particular, please review the following:

  • src/globalComponents.js - Understand how components are made global to the application. If you find yourself duplicating a block of code, it should probably be a component. Global components should not have strong coupling to data, make them flexible.
  • src/globalDirectives.js - Look for opportunities to create globally accessible directives for validation and other uses.
  • src/components/Dashboard/Views - Location for the bulk of the application components will live. Segment functionality into specific folders.
  • Maintenance App Docs on Sharepoint - Must be on the HC network to view - Breakdown of the applications

Github - We will be using formal pull requests.
